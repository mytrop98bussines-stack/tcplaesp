## 💡 NOTAS PARA EL DESARROLLADOR

### Cómo compartir el archivo:
1. **Por WhatsApp:** Enviar el archivo `index.html` como documento adjunto.
2. **Por email:** Adjuntar el archivo.
3. **Hosting gratuito (recomendado):**
   - **GitHub Pages:** Subir a un repositorio y activar Pages.
   - **Netlify:** Arrastrar la carpeta en netlify.com/drop
   - **Vercel:** Conectar repositorio y desplegar.

### Para generar un link compartible:
Si se sube a Netlify Drop:
1. Ir a https://app.netlify.com/drop
2. Arrastrar el archivo index.html
3. Copiar el link generado
4. Ese link es el que se comparte con los clientes

### Recomendación de hosting:
Netlify Drop es la opción más rápida y gratuita. No requiere cuenta ni configuración. El link generado funciona indefinidamente.
